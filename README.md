<h1 align="center">Vue-ssr</h1>

> Learn the Vue ssr.

## Build Setup

``` bash
# install dependencies
yarn install

# bundle client's file
yarn run build:client

# bundle server's file
yarn run build:server

# start the project
yarn start

```
